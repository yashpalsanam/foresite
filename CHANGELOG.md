# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Monorepo migration with Turborepo and pnpm
- Scoped package names: @foresite/admin-panel, @foresite/front-end, @foresite/back-end
- Comprehensive development scripts
- Docker support with docker-compose
- GitHub Actions CI/CD pipeline
- VS Code workspace configuration
- Prettier and ESLint configuration
- Contribution guidelines

### Changed
- Reorganized project structure into apps/ directory
- Updated build output directories
- Standardized environment variable configuration

### Fixed
- Port conflict between front-end and back-end (back-end now on 3001)

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- Admin Panel application
- Front-end website
- Back-end API
- Property management system
- User authentication
- Real-time notifications
- Image upload with Cloudinary
- Google Maps integration
