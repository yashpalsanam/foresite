# Foresite Admin Panel

Modern admin dashboard for Foresite real estate management platform.

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Axios
- Socket.io Client
- Firebase
- Chart.js

## Getting Started

### Installation

```bash
npm install
```

### Configuration

1. Copy `.env.example` to `.env`
2. Fill in all required environment variables

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
npm run preview
```

### Testing

```bash
npm test
npm run test:coverage
```

## Features

- User authentication
- Dashboard analytics
- Property management
- User management
- Inquiry tracking
- Real-time notifications
- Push notifications
- Theme switching
- PWA support
